# 2. long version

## Software Engineer (Full Stack)

## Location
San Francisco, CA / Remote (U.S.)

## Company Overview
We are a fast-growing technology company building modern, scalable software platforms used by thousands of customers worldwide. Our engineering team focuses on building reliable systems, intuitive user experiences, and high-performance backend services. We value clean architecture, strong collaboration, and engineers who enjoy solving real-world problems.

## Role Overview
We are looking for a **Full Stack Software Engineer** to join our engineering team. In this role, you will work across the entire stack—from building responsive front-end interfaces to designing scalable backend services and APIs. You will collaborate closely with product managers, designers, and other engineers to deliver high-quality features and improvements to our platform.

## Responsibilities
- Design, develop, and maintain full-stack web applications.
- Build responsive, user-friendly front-end interfaces.
- Develop scalable backend services and RESTful APIs.
- Collaborate with product managers and designers to translate product requirements into technical solutions.
- Write clean, maintainable, and well-tested code.
- Participate in code reviews and engineering discussions.
- Improve application performance, reliability, and scalability.
- Contribute to system architecture and technical decision-making.
- Troubleshoot production issues and implement fixes when needed.

## Required Qualifications
- Bachelor's degree in Computer Science, Software Engineering, or a related field (or equivalent experience).
- 3+ years of professional software development experience.
- Strong proficiency in **JavaScript/TypeScript** and modern web frameworks (React, Vue, or similar).
- Experience building backend services using **Node.js, Python, Java, or Go**.
- Familiarity with **REST APIs** and microservices architecture.
- Experience working with **SQL or NoSQL databases**.
- Understanding of version control systems such as **Git**.
- Ability to write maintainable, testable code and follow software engineering best practices.

## Preferred Qualifications
- Experience with **cloud platforms** such as AWS, GCP, or Azure.
- Familiarity with **Docker and containerized environments**.
- Experience with **CI/CD pipelines** and automated testing.
- Knowledge of system design and distributed systems.
- Experience working in Agile/Scrum development environments.

## Tech Stack (Example)
- Frontend: React, TypeScript, Vite
- Backend: Node.js / Python (FastAPI)
- Database: PostgreSQL, Redis
- Infrastructure: AWS, Docker, Kubernetes
- DevOps: GitHub Actions, CI/CD pipelines

## What We Offer
- Competitive salary and equity package
- Flexible remote work options
- Comprehensive health, dental, and vision insurance
- Professional development and learning opportunities
- Collaborative and inclusive engineering culture

## How to Apply
Please submit your resume, GitHub profile (if available), and a brief description of projects you’ve worked on that demonstrate your full-stack development experience.
